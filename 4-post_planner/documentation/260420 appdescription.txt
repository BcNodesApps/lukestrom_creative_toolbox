## 4-Post Tester – what it does

Generates a structured testing schedule for a song release
Outputs a ready-to-import `.ics` calendar file
Uses your calendar as the execution system (no thinking, just doing)

---

## Core flow

Start date = full video ready

Automatically plans:

* full video finalization
* DistroKid upload
* hook test (2 variants)
* moment test (2 variants)
* winner selection moments
* crossposting of best-performing combo
* repost cycles
* final cleanup

---

## Key features

Predefined, proven timing structure (no manual planning needed)
Reusable dropdown system for hooks and moments
Clear task naming (song + hook + moment combination)
Built-in ChatGPT prompts for captions where needed
Platform links included for direct posting
Exports clean `.ics` file for Outlook / calendar use

## Specialities (this is where it stands out)
